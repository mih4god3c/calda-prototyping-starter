# Branding

This folder holds all brand assets for this prototype. **Fill in `brand.md` before you start prototyping** — the AI reads this file to apply your colors, fonts, and style to every screen it generates.

## What Goes Here

| File / Folder | What to put there |
|---------------|------------------|
| `brand.md` | Colors, typography, radius, voice & tone |
| `logos/` | SVG or PNG versions of your logo (light + dark variants) |
| `fonts/` | Custom font files (.woff2 preferred) if not using Google Fonts |

## Quick Start

1. Open `brand.md` and fill in every section
2. Drop your logo into `logos/` (name it `logo.svg` or `logo-dark.svg`)
3. If you have custom fonts, drop `.woff2` files into `fonts/`
4. Tell Cursor: **"Apply our branding"** — it will read this folder and wire everything into the app

## Tips

- You don't need to have everything perfect before starting — even just a primary color makes a big difference
- Use [coolors.co](https://coolors.co) or [Realtime Colors](https://realtimecolors.com) to generate a palette
- For typography, Google Fonts has hundreds of options — just name the font family in `brand.md`
- The AI will remind you if it can't find a file it needs
